# ChatKit UI Scripts

This directory contains helper scripts for ChatKit UI integration.

## Available Scripts

### chatkit_helper.py
A command-line utility that provides code templates and guidance for setting up ChatKit integrations:

- Generates session endpoint code for backend
- Creates React component templates
- Provides HTML script tag for ChatKit
- Generates self-hosted server code
- Shows quick start guide

Run the script with:
```bash
python chatkit_helper.py
```

The script provides an interactive menu to select which code template you need for your ChatKit integration.
