# Contributing to Physical AI & Humanoid Robotics Book

Thank you for your interest in contributing to the Physical AI & Humanoid Robotics Book! We welcome contributions from the community to help improve this educational resource.

## How to Contribute

### Reporting Issues
- Use the GitHub Issues tab to report bugs, typos, or suggest improvements
- Be as specific as possible when describing the issue
- Include the section or page where the issue occurs

### Suggesting Improvements
- For content improvements, use the "Edit this page" link at the bottom of each page
- For larger changes or new content, open an issue first to discuss the proposed changes
- All contributions should align with the educational goals of the book

### Pull Requests
1. Fork the repository
2. Create a new branch for your changes
3. Make your changes following the existing style and structure
4. Submit a pull request with a clear description of your changes

## Content Guidelines

- Follow the existing document structure and formatting
- Ensure all code examples are accurate and well-commented
- Provide clear explanations suitable for beginner to intermediate learners
- Include relevant examples and practical applications

## Code of Conduct
Please be respectful and constructive in all interactions. We're all here to learn and improve our understanding of Physical AI and Humanoid Robotics.

## Questions?
If you have questions about contributing, feel free to open an issue for discussion.
