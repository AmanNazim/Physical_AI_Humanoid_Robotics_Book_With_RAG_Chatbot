# Physical AI & Humanoid Robotics Book with RAG Chatbot

This repository contains a comprehensive educational resource on Physical AI and Humanoid Robotics, deployed at: https://amannazim.github.io/Physical_AI_Humanoid_Robotics_Book_With_RAG_Chatbot/

## About the Book

This educational resource provides a complete learning path for understanding and working with Physical AI and Humanoid Robotics. The book is structured into four comprehensive modules:

1. **Module 1: ROS 2 Nervous System** - Covers the communication backbone for robotic systems
2. **Module 2: Digital Twin (Gazebo & Unity)** - Focuses on simulation environments for robot testing
3. **Module 3: AI-Robot Brain (NVIDIA Isaac)** - Explores AI reasoning and decision-making for robots
4. **Module 4: Vision-Language-Action (VLA)** - Integrates perception, understanding, and action systems

## Features

- Complete curriculum from beginner to advanced concepts
- Hands-on examples and exercises
- Industry-standard tools: ROS 2, Gazebo, Unity, NVIDIA Isaac
- Comprehensive assessments and projects
- Modern Docusaurus-based documentation site

## Deployment

This book is automatically deployed to GitHub Pages using GitHub Actions. The site is built and updated whenever changes are pushed to the main branch.
